# How to Install

Open terminal

Execute "git clone https://github.com/apolo1337/KaydyPurger"

Execute "cd KaydyPurger"

Execute "pip install -r requirements.txt"

Open "kaydyCFG.json" with any text editor!

Execute "py KaydyPurger.py"
